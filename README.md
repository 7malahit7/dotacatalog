# Dota Relics Catalog

Простой CRUD-каталог предметов Dota 2 с публичной страницей каталога, детальными страницами предметов и открытой админкой для управления записями.

## Возможности

- Главная страница со списком карточек предметов.
- Детальная страница для каждого предмета.
- Открытая админка без авторизации.
- Добавление, редактирование и удаление предметов.
- Хранение данных в SQLite-базе `data/catalog.db`.
- Базовый поиск и фильтр по редкости на странице каталога.

## Стек

- C++17
- HTML
- CSS
- JavaScript
- SQLite
- CMake

## Требования для Linux

Для Debian/Ubuntu-подобных систем:

```bash
sudo apt update
sudo apt install build-essential cmake sqlite3
```

Пакеты нужны для:

- `build-essential` - компилятор C++ и базовые инструменты сборки.
- `cmake` - сборка проекта.
- `sqlite3` - локальная SQLite-база данных.

## Запуск

Собрать и запустить сервер:

```bash
cmake -S . -B build
cmake --build build
./build/dota_relics_catalog
```

Также можно запускать из папки сборки:

```bash
cd build
./dota_relics_catalog
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8080
```

## Основные страницы

- `http://127.0.0.1:8080/` - каталог предметов.
- `http://127.0.0.1:8080/items/1` - детальная страница предмета.
- `http://127.0.0.1:8080/admin` - админка.
- `http://127.0.0.1:8080/admin/new` - добавление предмета.
- `http://127.0.0.1:8080/admin/edit/1` - редактирование предмета.

## Проверка CRUD

1. Открыть `http://127.0.0.1:8080/admin`.
2. Нажать `Добавить предмет` и сохранить новую запись.
3. Проверить, что запись появилась в таблице и на главной странице.
4. Открыть редактирование записи и изменить данные.
5. Удалить запись через кнопку `Удалить`.

Чтобы сбросить данные к стартовым демо-предметам, остановите сервер и удалите файл `data/catalog.db`.
